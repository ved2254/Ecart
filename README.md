# 🛒 MERN E-Commerce Application

A full-stack E-Commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This project simulates a real-world online shopping platform with secure authentication, cart management, order processing, and payment integration.

---

## 🚀 Features

- 🔐 JWT Authentication (Login / Register)
- 🔒 Password Hashing using bcrypt
- 🛍 Product Listing & Product Details Page
- 🛒 Add to Cart & Cart Persistence (Redux)
- 📦 Order Placement System
- 💳 Razorpay Payment Integration (Test Mode)
- 🛠 Admin Product Management
- 🌐 RESTful API Architecture
- 📁 Image Upload Support
- 🧾 Order History Tracking

---

## 🏗 Tech Stack

### Frontend:
- React.js
